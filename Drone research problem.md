# Drone research problem

**AI-Powered Immersive Virtual Reality Drone Simulation Using Real Flight Dataset**
Existing AI-based drone simulation models focus mainly on predictive accuracy using machine learning techniques such as CNNs; however, they lack real-time immersive simulation environments and human-in-the-loop interaction. Current studies do not integrate AI-based drone dynamics models with virtual reality environments to enable interactive pilot training and qualitative validation. Therefore, there is a research gap in developing a real-time VR-based drone simulation system that integrates AI-driven flight dynamics with immersive visualization and high-performance computing for enhanced training and evaluation.
**Main ****Goal**
To develop a real-time AI-based VR drone simulation system.
**Research**** Objectives**
- To improve CNN-based drone flight prediction model
- To integrate drone dynamics model with Unity VR environment
- To develop real-time drone flight simulation
- To enable pilot interaction using VR headset
- To evaluate simulation accuracy and user experience
- To compare VR simulation with real drone flight data
